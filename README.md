# msgpack-serialization

This is an implementation of the MessagePack serialization format. Read the [MessagePack specification](https://github.com/msgpack/msgpack/blob/master/spec.md) and browse [their website](https://msgpack.org/) for more information about MessagePack.

I decided to write this MessagePack implementation for my moomoo.io script, since moomoo.io uses MessagePack for efficient network communication.
At the moment, my moomoo.io script uses a msgpack library I found on NPM, but I want to use my own libraries, since using libraries created by other people is like skidding in a way.
